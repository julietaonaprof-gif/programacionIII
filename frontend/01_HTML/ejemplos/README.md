# Curso de HTML — Nivel Básico

Material de estudio de HTML con archivos comentados que cubren desde la estructura básica de un documento hasta etiquetas semánticas, enlaces, imágenes y listas.

## Requisitos

- Un navegador web moderno (Chrome, Firefox, Edge, Safari).
- No se necesita instalar nada. No requiere servidor, base de datos, ni dependencias.
- Opcionalmente, un editor de código como [Visual Studio Code](https://code.visualstudio.com/).

## Cómo ejecutar los archivos

Cada archivo es independiente. Se puede abrir de cualquiera de estas formas:

**Doble clic** — Hacé doble clic sobre cualquier archivo `.html` y se abre en tu navegador.

**Arrastrar al navegador** — Arrastrá el archivo a una ventana abierta de Chrome/Firefox.

**Live Server (VS Code)** — Instalá la extensión [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer), click derecho sobre el archivo → "Open with Live Server". Tiene la ventaja de que se recarga solo cuando guardás cambios.


## Contenido

Los archivos están numerados en orden de aprendizaje:

## Orden de estudio recomendado

1. Empezá por el **archivo 1** para entender la estructura base de todo documento HTML.
2. Seguí en orden numérico — cada archivo supone que entendés los anteriores.
3. Los archivos están llenos de **comentarios** que explican cada etiqueta, sus reglas, cuándo usarla y cuándo no, con links a la documentación oficial. Es importante que puedas leer esos comentarios, abrir los documentos, con tu IDE o editor de texto.
4. Modificá los archivos, experimentá, rompé cosas y volvé a armarlas.
