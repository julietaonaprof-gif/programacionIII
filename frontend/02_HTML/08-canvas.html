Guía de <canvas> en HTML

<canvas> es un contenedor en blanco donde se dibujan gráficos, animaciones o juegos usando JavaScript. La etiqueta sola no dibuja nada.
📖 MDN — <canvas>

1. Canvas básico

<canvas> crea un área de dibujo. Sin JavaScript, es un rectángulo vacío (invisible sin borde).

Tu navegador no soporta canvas. 
<canvas id="canvas1" width="400" height="200">
  Tu navegador no soporta canvas.
</canvas>

<script>
  var canvas = document.getElementById('canvas1');
  var ctx = canvas.getContext('2d');
  ctx.fillStyle = '#3498db';
  ctx.fillRect(50, 50, 300, 100);
</script>

2. Figuras básicas — Rectángulos

Los rectángulos son las únicas figuras que canvas dibuja directamente. Todo lo demás se hace con "paths" (trazados). 

// Rectángulo relleno
ctx.fillStyle = '#3498db';
ctx.fillRect(20, 20, 120, 80);

// Rectángulo solo borde
ctx.strokeStyle = '#e74c3c';
ctx.lineWidth = 3;
ctx.strokeRect(170, 20, 120, 80);

// Borrar un área
ctx.clearRect(350, 50, 90, 90);

3. Líneas y trazados (paths)

Todo lo que no sea un rectángulo se dibuja con paths: empezás un trazo, definís puntos, y después lo rellenás o dibujás el borde. 

// Línea simple
ctx.beginPath();
ctx.moveTo(20, 100);
ctx.lineTo(200, 30);
ctx.stroke();

// Triángulo relleno con borde
ctx.beginPath();
ctx.moveTo(250, 150);
ctx.lineTo(350, 30);
ctx.lineTo(450, 150);
ctx.closePath();
ctx.fill();
ctx.stroke();

4. Círculos y arcos

No existe un método "circle". Se usa arc() para dibujar arcos y círculos completos. 

// Círculo relleno
ctx.beginPath();
ctx.arc(80, 120, 60, 0, Math.PI * 2);
ctx.fillStyle = '#e74c3c';
ctx.fill();

// Círculo solo borde
ctx.beginPath();
ctx.arc(230, 120, 60, 0, Math.PI * 2);
ctx.stroke();

// Semicírculo
ctx.beginPath();
ctx.arc(380, 120, 60, 0, Math.PI);
ctx.fill();

5. Texto

// Texto relleno
ctx.font = '30px Arial';
ctx.fillStyle = '#2c3e50';
ctx.fillText('¡Hola Canvas!', 20, 60);

// Texto solo borde
ctx.font = 'bold 40px Georgia';
ctx.strokeText('Texto con borde', 20, 130);

// Texto centrado
ctx.textAlign = 'center';
ctx.fillText('Centrado', 250, 180);

6. Colores y gradientes

// Gradiente lineal
var grad = ctx.createLinearGradient(0, 0, 220, 0);
grad.addColorStop(0, '#3498db');
grad.addColorStop(1, '#e74c3c');
ctx.fillStyle = grad;
ctx.fillRect(10, 10, 220, 80);

// Gradiente radial
var grad2 = ctx.createRadialGradient(370, 100, 10, 370, 100, 80);
grad2.addColorStop(0, '#f1c40f');
grad2.addColorStop(1, '#e67e22');
ctx.fillStyle = grad2;

// Transparencia
ctx.fillStyle = 'rgba(46, 204, 113, 0.5)';

7. Dibujar imágenes

Se pueden dibujar imágenes dentro del canvas con drawImage(). 

var img = new Image();
img.onload = function() {
  ctx.drawImage(img, 10, 10);             // Posición
  ctx.drawImage(img, 280, 10, 200, 180);  // Redimensionada
};
img.src = 'imagen.jpg';

8. Animación simple

Las animaciones se hacen redibujando el canvas muchas veces por segundo con requestAnimationFrame(). 

var x = 0;

function animar() {
  ctx.clearRect(0, 0, canvas.width, canvas.height);  // Borrar

  ctx.beginPath();
  ctx.arc(x, 100, 20, 0, Math.PI * 2);               // Dibujar
  ctx.fill();

  x += 2;                                              // Mover
  if (x > canvas.width + 20) x = -20;                 // Rebote

  requestAnimationFrame(animar);                       // Siguiente frame
}
animar();

9. Interacción con el mouse

Canvas detecta eventos del mouse como cualquier otro elemento HTML. Pero hay que calcular la posición relativa al canvas.

Hacé clic en el canvas para dibujar círculos. Mové el mouse para ver la posición.

canvas.addEventListener('click', function(evento) {
  var rect = canvas.getBoundingClientRect();
  var x = evento.clientX - rect.left;
  var y = evento.clientY - rect.top;

  ctx.beginPath();
  ctx.arc(x, y, 15, 0, Math.PI * 2);
  ctx.fill();
});

10. Accesibilidad

Canvas es inaccesible por defecto: los lectores de pantalla no pueden leer lo que hay dibujado. Hay que agregar alternativas.



Gráfico de barras: Enero $100, Febrero $150, Marzo $200.
Gráfico de barras que muestra las ventas del primer trimestre. Enero: $100. Febrero: $150. Marzo: $200. La tendencia es creciente. 
<!-- Opción 1: aria-label -->
<canvas role="img"
        aria-label="Gráfico: Enero $100, Febrero $150, Marzo $200">
</canvas>

<!-- Opción 2: aria-describedby con texto oculto -->
<canvas role="img"
        aria-label="Gráfico de ventas"
        aria-describedby="descripcion">
  <p>Fallback: Enero $100, Febrero $150, Marzo $200.</p>
</canvas>
<div id="descripcion" hidden>
  Descripción larga del gráfico...
</div>

Resumen

Atributos de <canvas>
Atributo Valores Descripción
width px (default: 300) Ancho del área de dibujo
height px (default: 150) Alto del área de dibujo
id texto Para acceder desde JavaScript

Métodos principales del contexto 2D
Método Descripción
fillRect(x, y, w, h) Rectángulo relleno
strokeRect(x, y, w, h) Rectángulo solo borde
clearRect(x, y, w, h) Borrar un área
beginPath() Empezar un trazo nuevo
moveTo(x, y) Mover el lápiz sin dibujar
lineTo(x, y) Línea hasta un punto
arc(x, y, r, ini, fin) Arco o círculo
closePath() Cerrar el trazo
fill() Rellenar el trazo
stroke() Dibujar el borde del trazo
fillText(txt, x, y) Texto relleno
strokeText(txt, x, y) Texto solo borde
drawImage(img, x, y) Dibujar imagen
createLinearGradient() Gradiente lineal
createRadialGradient() Gradiente radial

Propiedades principales del contexto 2D
Propiedad Descripción
fillStyle Color de relleno (color, gradiente, patrón)
strokeStyle Color de borde
lineWidth Grosor de línea (en px)
font Fuente del texto (formato CSS)
textAlign Alineación horizontal del texto
textBaseline Alineación vertical del texto
globalAlpha Transparencia global (0 a 1)
lineCap Forma del extremo de línea (butt, round, square)
lineJoin Forma de la unión de líneas (miter, round, bevel)

Buenas prácticas

  1. Poner width y height como atributos, no como CSS. Si usás CSS, el canvas se estira y se ve pixelado.
  2. Siempre agregar texto de fallback entre las etiquetas para navegadores sin soporte.
  3. Usar role="img" y aria-label para describir lo que se dibuja.
  4. Usar requestAnimationFrame() para animaciones, nunca setInterval() (es menos eficiente).
  5. Borrar el canvas antes de redibujar en animaciones con clearRect().
  6. Para gráficos de datos usar una librería como Chart.js en vez de dibujar a mano. Canvas puro es para casos personalizados.
  7. Canvas es para gráficos dinámicos. Para gráficos estáticos, usar SVG (es accesible, escalable y se puede estilizar con CSS).
