Guía de <iframe> en HTML

<iframe> incrusta otra página web dentro de tu página. Es un "marco" que muestra contenido externo.
📖 MDN — <iframe>

1. Iframe básico

<iframe src="https://es.wikipedia.org/wiki/HTML"
        width="600" height="400"
        title="Artículo de Wikipedia sobre HTML">
</iframe>

2. Incrustar YouTube

La URL de embed de YouTube es diferente a la URL normal.
Normal: https://www.youtube.com/watch?v=VIDEO_ID
Embed: https://www.youtube.com/embed/VIDEO_ID 

<iframe width="560" height="315"
        src="https://www.youtube.com/embed/VIDEO_ID"
        title="Video de YouTube"
        allowfullscreen
        loading="lazy">
</iframe>

YouTube con parámetros

<!-- Empieza en segundo 10, sin videos relacionados, con subtítulos -->
<iframe width="560" height="315"
        src="https://www.youtube.com/embed/VIDEO_ID?start=10&rel=0&cc_load_policy=1"
        title="Video con parámetros"
        allowfullscreen>
</iframe>

<!-- Autoplay silenciado en loop -->
<iframe width="560" height="315"
        src="https://www.youtube.com/embed/VIDEO_ID?autoplay=1&mute=1&loop=1"
        title="Video en loop"
        allowfullscreen>
</iframe>

3. Incrustar Google Maps

En Google Maps: Menú → Compartir → Insertar un mapa → copiar el iframe. 

<iframe width="600" height="400"
        src="https://www.google.com/maps/embed?pb=..."
        title="Mapa del Obelisco, Buenos Aires"
        loading="lazy"
        allowfullscreen>
</iframe>

4. Incrustar Spotify

En Spotify: botón Compartir → Copiar código de inserción. 

<iframe width="300" height="380"
        src="https://open.spotify.com/embed/track/TRACK_ID"
        title="Canción en Spotify"
        allow="encrypted-media"
        loading="lazy">
</iframe>

5. Incrustar una página propia

<!-- Otra página de tu propio sitio -->
<iframe src="otra-pagina.html"
        width="100%" height="300"
        title="Página de contacto">
</iframe>

<!-- URL externa -->
<iframe src="https://ejemplo.com"
        width="100%" height="400"
        title="Sitio externo">
</iframe>

6. Carga diferida — loading="lazy"

loading="lazy" hace que el iframe se cargue solo cuando el usuario hace scroll hasta él. Mejora la velocidad de carga de la página. 

<iframe src="https://www.youtube.com/embed/VIDEO_ID"
        width="560" height="315"
        title="Video"
        loading="lazy"
        allowfullscreen>
</iframe>

7. Seguridad — sandbox

sandbox restringe lo que puede hacer el contenido del iframe. Es como ponerlo en una caja de seguridad.

sandbox (bloquea todo):



sandbox con permisos (permite scripts y forms):

<!-- Bloquea TODO -->
<iframe src="https://ejemplo.com" sandbox
        title="Restringido total">
</iframe>

<!-- Permite scripts y formularios -->
<iframe src="https://ejemplo.com"
        sandbox="allow-scripts allow-forms"
        title="Permite scripts y forms">
</iframe>

<!-- Permite scripts, forms y popups -->
<iframe src="https://ejemplo.com"
        sandbox="allow-scripts allow-forms allow-popups"
        title="Permite scripts, forms y popups">
</iframe>

8. Permisos de funcionalidades — allow

allow controla qué APIs del navegador puede usar el iframe (cámara, micrófono, geolocalización, etc.). 

<!-- Videollamada: necesita cámara y micrófono -->
<iframe src="https://ejemplo.com/videollamada"
        width="600" height="400"
        title="Videollamada"
        allow="camera; microphone; fullscreen">
</iframe>

<!-- YouTube: permisos típicos -->
<iframe src="https://www.youtube.com/embed/VIDEO_ID"
        allow="accelerometer; autoplay; clipboard-write;
               encrypted-media; gyroscope; picture-in-picture"
        allowfullscreen>
</iframe>

9. Privacidad — referrerpolicy

Controla cuánta información de tu página se envía al sitio del iframe. 

<!-- No envía información de tu página al sitio del iframe -->
<iframe src="https://ejemplo.com"
        referrerpolicy="no-referrer"
        title="Privado">
</iframe>

<!-- Solo envía tu dominio, no la URL completa -->
<iframe src="https://ejemplo.com"
        referrerpolicy="origin"
        title="Solo dominio">
</iframe>

10. Sitios que bloquean iframe

No todos los sitios se pueden incrustar. Muchos bloquean los iframes por seguridad usando el header X-Frame-Options o Content-Security-Policy.

Esto NO funciona (Google bloquea iframes):

<!-- NO funciona: Google bloquea iframes -->
<iframe src="https://www.google.com" title="Bloqueado"></iframe>

<!-- SÍ funciona: YouTube tiene URL de embed -->
<iframe src="https://www.youtube.com/embed/VIDEO_ID" title="YouTube"></iframe>

<!-- SÍ funciona: Google Maps tiene URL de embed -->
<iframe src="https://www.google.com/maps/embed?pb=..." title="Mapa"></iframe>

Resumen de atributos de <iframe>

Todos los atributos de la etiqueta iframe
Atributo Valores Descripción
src URL Página a incrustar
width px o % Ancho del marco
height px Alto del marco
title texto Descripción para accesibilidad (obligatorio)
loading eager | lazy Cuándo cargar el iframe
sandbox permisos separados por espacio Restringe lo que puede hacer el iframe
allow permisos separados por ; APIs del navegador permitidas
allowfullscreen (booleano) Permite pantalla completa
referrerpolicy no-referrer, origin, etc. Qué info de tu página se envía
name texto Nombre del iframe (para target de links)
srcdoc HTML inline HTML directo en vez de una URL

Buenas prácticas

  1. Siempre poner title para accesibilidad.
  2. Usar loading="lazy" en iframes que no se ven al cargar.
  3. Usar sandbox cuando incrustes contenido que no controlás.
  4. No incrustar sitios que no confías: el iframe puede ejecutar código.
  5. Usar las URLs de embed, no las URLs normales (YouTube, Vimeo, Maps, Spotify).
  6. No abusar de iframes: cada uno carga una página completa. Muchos iframes = página lenta.
