# Frontend

Curso completo de desarrollo frontend. Cada carpeta es una clase con archivos comentados, ejercicios prácticos y ejemplos funcionales.

## Cómo usar este repositorio

1. Entrá a la carpeta de la clase que quieras estudiar.
2. Cada carpeta tiene su propio `README.md` con instrucciones específicas.
3. Los archivos `.html` se abren con doble clic en el navegador o con Live Server en VS Code.

## Clases

| # | Carpeta | Tema | Estado |
|---|---------|------|--------|
| 1 | `01_HTML/` | HTML: estructura, head, textos, semántica, enlaces, imágenes, listas | ✅ |
| 2 | `02_HTML/` | HTML: formularios, tablas, dialog, ejercicios formularios y tablas | ✅ |


> Los temas y la cantidad de clases pueden cambiar a medida que avance el curso.
