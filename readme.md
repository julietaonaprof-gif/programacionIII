# Repositorio de Ejemplos de Clase

Este repositorio contiene los ejemplos de código que se desarrollan a lo largo de la cursada, organizados por capas de aplicación para facilitar su consulta y ejecución.

## 📂 Estructura del Proyecto

El repositorio se divide en dos directorios principales:

* **`/backend`**: Ejemplos relacionados con servidores, APIs y bases de datos.
* **`/frontend`**: Ejemplos de interfaces de usuario y lógica de cliente.

---

## 🛠️ Tecnologías Utilizadas

### Backend
* **Node.js**: Entorno de ejecución para JavaScript.
* **TypeScript**: Superset de JavaScript que añade tipado estático.
* **Express.js**: Framework para la creación de APIs web.
* **SQLite**: Motor de base de datos relacional ligero.

### Frontend
* **HTML5 & CSS3**: Estructura y diseño visual.
* **JavaScript (ES6+)**: Lógica interactiva en el navegador.
* **React.js**: Biblioteca para la construcción de interfaces de usuario modernas.

---

## 📋 Pre-requisitos

Antes de clonar el repositorio y ejecutar los ejemplos, asegúrate de cumplir con los siguientes requisitos:

1.  **Node.js**: Es fundamental tener instalado Node.js (se recomienda la versión LTS) para gestionar paquetes y ejecutar el entorno de backend.
    * [Descargar Node.js aquí](https://nodejs.org/)
2.  **Gestor de paquetes**: Al instalar Node.js, se incluirá automáticamente `npm`.
3.  **Git**: Para clonar el repositorio en tu máquina local.

---
